Web Chat Server

I created a basic WebChat server that allows users to communicate online. I created a back-end server that receives Web Socket requests, decodes them, and sends the response to the appropriate user. I created the front-end HTML, CSS, and JavaScript files to accommodate the user experience.

DNS Server

I built a DNS server that receives DNS requests, parses the message information, and if the request is already contained in the program cache the appropriate message is sent back. If the request doesn't exist in the cache, the message is forwarded to Google and Google's response is sent to the client.
